To run this simulation two possibilities :

	1) Using command lines
		i) Go to this directory
		ii) Execute in your Terminal "Rscript convergence_simu.R"
	2) Open R or Rstudio (define working directory at location of this file) and run convergence_simu.R

Output : print Table 1 of Section 3.1.1 in console. 

Remark : parameters are set to generate the exact same table. To be faster, decrease "nb_application_of_mgcca_algo" in "convergence_simu.R"
